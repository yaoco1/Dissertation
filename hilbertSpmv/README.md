# Hilbert Curve Reordering for GPU Sparse Matrix-Vector Multiplication: 
# A Comparative Study of Row, Blocked, and Global Permutation Across CUDA Kernel Implementations

## 1. Overiew
This project implements three GPU Sparse Matrix-Vector Multiplication (SpMV) kernels and three Hilbert curve reordering strategies for CSR-format sparse matrices. 
All kernels are benchmarked against NVIDIA cuSPARSE across seven sparse matrices spanning graph, science computing, random sparse, and large-scale graph.

## 2. Project Structure
.
├── src/
│   ├── naiveCsr.cu            # Scalar CSR kernel (one thread per row)
│   ├── warpCsr.cu             # Warp-cooperative CSR kernel
│   ├── sharedCsr.cu           # Shared memory CSR kernel
│   ├── baseline.cu            # cuSPARSE benchmark
│   ├── rowReorder.cu          # Row reordering benchmark
│   ├── blockedHilbert.cu      # Blocked Hilbert reordering benchmark
│   ├── globalHilbert.cu       # Global Hilbert reordering benchmark
│   ├── reorder.cu             # Shared reordering utilities (load, kernels, helpers)
│   └── reorder.h              # ReorderResult struct and function declarations
├── data/
│   ├── sciComputingMatrix/    # cant, ct20stif, nlpkkt80
│   ├── randomSparseMatrix/    # barrier2-1, nlpkkt80
│   ├── graphMatrix/           # roadNet-CA, roadNet-TX
│   └── largeScaleGraphMatrix/ # com-Youtube
├── build/                     # Compiled binaries (generated by make)
├── test/
│   └── allTests.sh            # Automated test script for all kernels × matrices
└── Makefile

## 3. Build
make          # Build all targets
make clean

## 4. Usage
Baseline kernels (no reordering):
./build/naiveCsr   <matrix.mtx>
./build/warpCsr    <matrix.mtx>
./build/sharedCsr  <matrix.mtx>
./build/baseline   <matrix.mtx>

Reordering strategies:
./build/rowReorder     <matrix.mtx>
./build/blockedHilbert <matrix.mtx> [blockSize]   # default blockSize=512
./build/globalHilbert  <matrix.mtx> [blockSize]   # default blockSize=512

Run all tests:
bash test/allTests.sh
